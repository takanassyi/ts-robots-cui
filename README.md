# ts-robots-cui

![result](https://github.com/takanassyi/ts-robots-cui/blob/main/img/robots.gif)

## Prerequisites
- Node.js 12.19.0 or above
- npm 6.14.8 or above

## How to build and start "Robots"

1. Clone or download this project
2. `npm install`
3. `tsc`
4. `node ./dist/index.js`
